Privacy Policy of Savings Manager

What information does this application collect?

This application does not collect any information related to you. All your financial information that you provide in this application is stored only at your device, where this application is operating. No information is being transferred over internet.

However, this application provides backup functionality, which transfers user’s data to Google DriveTM account provided by the user. Data protection on Google DriveTM is governed under Google’s Privacy Policy.
Note: Data Backup functionality is totally user initiated. Without user consent no data is being backed up or transferred to Google DriveTM over internet.

There may be third party services (e.g. Google Play Services, Ad services etc.) embedded in the application, which may collect information from you; however, they do not have access to your financial data stored in this application.

Your financial data is protected in the application using strong security algorithms.

Does this application use any cookies or store any data on your device?

This application does not use cookie at all.

Third party services

There may be other third party services embedded in this application, as stated below:
- Google Play Services – Google Drive (Privay Policy)

Which app permissions does this application ask from user?

This application does not ask for any sensitive permission from user while installing this application from the "Market". However, this application may ask for the below permissions:
- Internet connectivity
- Network Information (e.g network type-wifi/mobile data, network speed etc.)
- Read and Write on Device Storage (Needed to save internal copy of user data securely.)
- Run at startup (To setup Notifications & Reminders)
